# Student Performance Tracker

##  Overview

This project is a web-based application that helps students track and analyze their academic performance. It allows users to add subjects and marks, view statistics, and visualize performance using charts.

##  Features

* Add and delete subjects with marks
* Automatic calculation of average, highest, and lowest scores
* Performance status (Excellent / Average / Needs Improvement)
* Graphical visualization using Chart.js
* Weak subject highlighting
* Data stored using LocalStorage

##  Technologies Used

* HTML
* CSS
* JavaScript
* Chart.js

##  How to Run

1. Download or clone the repository
2. Open index.html in any browser

##  Learning Outcomes

* DOM manipulation
* Data handling using arrays
* LocalStorage usage
* Basic data visualization
